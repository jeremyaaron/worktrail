The app

A reference workflow with enough product surface to matter.

Worktrail is intentionally small, but it is not a toy shell. The v0.0.2 app covers project settings, label catalogs, keyed work items, lists, a drag/drop board, comment lifecycle controls, local actor selection, and activity events so implementation choices have real consequences.

Projects

Configurable projects with readable lifecycle state.

Edit metadata, manage archive/reactivate flows, and keep archived projects read-only.

Work items

Stable references and complete work item forms.

Create keyed work items with type, status, priority, assignee, labels, due dates, estimates, and comments.

Board

Drag/drop movement with a status-menu fallback.

Move cards with Angular CDK drag/drop while server-side workflow validation remains authoritative.

Collaboration

Comments, tombstones, and activity history.

Add, edit, and delete comments with visible tombstones and lifecycle activity.

v0.0.2 capabilities

Enough product depth to exercise real patterns.

The current sprint strengthens the app around adoptability: project keys, immutable work item display keys, project label administration, label assignment on create, archived-label handling, archived-project read-only behavior, comment edit/delete, deleted-comment tombstones, and broader activity coverage.

Architecture

Local-first now. Cloud-shaped from the first cut.

The MVP runs locally with Angular, Express, and Postgres. The internal boundaries are deliberately boring: shared DTO contracts, request validation, repository-backed services, actor context, command-style writes, and endpoint handlers that can outlive the local Express adapter.

Angular SPAstatic build suitable for S3 and CloudFront

Express adapterlocal runtime only for v0.0.2
Postgres + Drizzlecommitted SQL migrations and deterministic seed data
Playwright smokebrowser-level workflow coverage for release readiness

Current scope

Local-first, cloud-shaped, still honest about limits.

Worktrail v0.0.2 is still a local reference app. Production authentication, custom workflows, persisted board ordering, notifications, attachments, imports, and hosted infrastructure are intentionally out of scope. The value today is a working product surface that can accumulate evidence before patterns are extracted into broader framework ideas.
